About the game

Circuit runner is a web+circuit toolkit which help kids understand basic knowledge of electronic circuits. It a cooperative game which allows kids to discuss and explore how the circuits work.

  1. Power
  2. LED
  3. Resistor
  4. Capacitor

Power

Power is both the start and the end, it gives you the energy to run! When you get start from the power, you can run 3 steps each time.

LED

LED is the light point, you should get as more LEDs as possible. If you get more LEDs than your friend, you win.

Resistor

Resistor will resist the flow of the electric. So if you encounter a resisitor, you'll be blocked for one round.

Capacitor

Capacitor stores energy. Everytime you get to a capacitor, you can double the steps you can run next round.
